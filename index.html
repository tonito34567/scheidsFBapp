⚙️ Configuratie vereist

  1. Ga naar script.google.com → Nieuw project
  2. Plak de inhoud van Code.gs (zie bijgeleverd bestand)
  3. Maak een Google Sheet aan, kopieer de Sheet ID uit de URL
  4. Vul de SHEET_ID in bovenaan het script
  5. Voer functie initSheets() eenmalig uit
  6. Deploy → Web App → Execute as Me → Anyone with link
  7. Kopieer de Web App URL en plak hieronder
